On field standard size 9 x 9 every step appear 3 balls random color. At all, in game 7 different colors of ball,you must build it at horizontal or vertical line. Balls are delete after 4 or more balls at line. For each delete balls you get scores( for 4 balls - 8, 5-10, 6-18, 7-21, 8-32, 9-36, 10-50, 11-55 ). After balls delete - appear random ball postponed to next move.On bottom side of game field you can choose field-size and can switch on hints. If field size 7x7 for delete enough 3 balls same color build on line.After each step appears 2 random balls. If field size 11x11, balls delete after 5 and more same color build on line and after steps appear 4 random balls.A task - score more points. Good luck!
